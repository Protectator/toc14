A string in A cannot begin with a zero; it needs to switch to 1s eventually in order to have an even number of 1s, but "01" is banned. Also, since "01" is banned, all 1s must precede all 0s. Therefore, A can be redefined as the concatenation of

\begin{align*}
A_1 \subseteq \{1\}^* &:= \{ \omega \  |\  \omega \text{ contains an odd number of 1s} \} \\
A_2 \subseteq \{0\}^* &:= \{ \omega \  | \  \omega \text{ contains an even number of 0s} \}. 
\end{align*}

Both $A_1$ and $A_2$ are regular, because we can construct the following DFAs for them:

Since A is the concatenation of two regular languages ($A_1 \circ A_2$), it is also regular as we saw in the class.
